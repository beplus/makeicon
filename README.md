## Installation

downlouad and unzip 

or

```bash
$ brew install makeicon
```

## Usage
#### Command-Line Interface
##### Help
```bash
$ makeicon --help
```
or 
```bash
$ makeicon -h
```

##### Make
for create directory with icons from file `Image.png`
```bash
$ makeicons --file Image.png
```
or 
```bash
$ makeicons -f Image.png
```
